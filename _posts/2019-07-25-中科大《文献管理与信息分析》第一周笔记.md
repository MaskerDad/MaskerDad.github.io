# 中科大《文献管理与信息分析》第一周笔记——信息社会的挑战及课程内容设计

## 问题1：科研到底是什么？

- 科研只有第一，没有第二
- 科学研究需要探索未知，面对新事物（恐惧感）
- 需要解决新问题
- 科研需要大量的重复（如何看待重复？）
- 科研会经历多次失败

## 问题2：科研有什么意义？

1. 全球视野，以及发现需求的眼光
2. 面对未知事物的勇气，以及愈挫愈勇的韧性
3. 解决问题的能力
4. 在持续努力和联系过程中形成习惯
5. 任何成功的人都是优秀的研究者！

## 问题3：什么能够让人快速成长？

- **科研** 
- **创业**

## 问题4：好的选题从何而来？

1. 知己知彼：你对一个领域的了解程度会直接影响你的选题
2. 现实需求：真正来自现实生活中的需求最有生命力
3. 长期思考：有的问题需要多年的思考
4. 宽阔胸怀：一个人的出发点直接决定了选题可能的高度
5. 好的选题绝非一日之功

## 问题5：本课程的目标是什么？

1. 提升快速学习的能力
2. 提升科学研究的能力
3. 提升解决问题的能力
4. 提升洞察力和创新力

## 问题8：高效学习的策略是什么？

### 策略一：专注

 同样的努力，不同的结果

可怕的不是我们有没有缺点，而是我们没有优点

### 策略二：向牛人学习

所有的学习都是向他人学习

- 名师出高徒，学习对象的水平决定了你的高度
- 不同的知识载体，传递信息的效率不同
- 同类知识载体中，不同质量的信息价值不同（阅读高质量的文献）
- 快速找到高质量的信息是快速学习的基础

### 策略三：善假于物

君子生非异也，善假于物也。《荀子·劝学》

### 策略四：投资时间

**时间投资**：花一定的时间学习，提升效率，从而节约更多的时间。

当自己很忙很累很辛苦的时候，要有意识去思考如何改变。

### 策略五：分享知识

建立分享意识

- 分享是高效、积极、主动的学习
- 分享是学习的动力所在（输出带动输入）
- 分享可以为自己带来更多的机会
- 分享可以建立自己的品牌，扩大人脉资源
- 分享科技检验需求，身边每一个牛叉的人都是积极地分享者

### 总结

1. 有所学有所不学的专注意识
2. 学习精华内容（牛人）的意识
3. 善假于物的工具意识
4. 提升效率的时间投资意识
5. 输出带动输入的分享意识
